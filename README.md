# youtube-rip

## Introduction
`youtube-rip` is a small AppleScript GUI program that uses `yt-dlp` to rip the audio from YouTube videos and save them locally, allowing you to save songs and podcasts locally with ease.

## Installation
`youtube-rip` depends on yt-dlp installed via Homebrew.

Download Homebrew (Mac):
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Download `yt-dlp`: 
```bash
brew install yt-dlp
```

Download the .app ZIP:

[Release (GitHub)](https://github.com/mr-sunset/youtube-rip/releases/tag/v1.0.0)
